Objetivo

Adquirir destrezas y habilidades que le permitan desarrollar sitios webs dinámicos, en donde el usuario interactue con información en base de datos MySQL a través de PHP.

Temario

    Modulo 1

  • Lenguaje PHP Introducción a PHP y HTML.
  • Sintaxis básica.
  • Condicionales y control de flujo (IF/ELSE, SWITCH).
  • Manejo de bucles (DO, DO-WHILE).
  • Librerias estandar.
  • Formularios
  • Funciones.

    Modulo 2

  • Gestor de base de datos MySQL
  • Introducción.
  • Manejo (creación de base de datos, usuarios, consulta y modificación de datos).
  • Administración.
  • Optimización.

    Modulo 3

  • Interconexión MySQL con PHP
  • Acceso desde PHP a MySQL.
  • Autenticación de usuarios.
  • Modificación de bases de datos y tablas desde PHP.
  • Seguridad
